# **App Name**: Fridge Genie

## Core Features:

- Ingredient Input: Allow users to input the ingredients available in their fridge.
- Recipe Generation: Generate recipe suggestions based on the inputted ingredients, utilizing a tool that assesses ingredient compatibility and suggests complementary items to consider.
- Recipe Display: Display detailed instructions for each recipe, including steps, cooking times, and nutritional information.
- Favorite Recipes: Allow users to save their favorite recipes for later access.

## Style Guidelines:

- Primary color: Fresh green (#8FBC8F) to evoke a sense of freshness and natural ingredients.
- Background color: Off-white (#F5F5DC) to provide a clean and user-friendly interface.
- Accent color: Light olive (#BDB76B) for subtle highlights and call-to-action elements.
- Body and headline font: 'PT Sans' for a modern and warm feel, suitable for both headlines and body text.
- Use minimalist icons related to food and cooking, with a thin stroke style.
- Simple and clean layout with clear sections for ingredient input, recipe suggestions, and detailed instructions.
- Subtle transitions and animations for loading new recipes and navigating between sections.
